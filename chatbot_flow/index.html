Fluxo

botStartChat

Inicia o Chatbot

Funcao:
botStartChat()

Exemplo: 
    <?php

        header("Access-Control-Allow-Origin: *");

        error_reporting(1);

        $id=$argv[1];
        $telefone=$argv[2];
        $assunto=$argv[3];


        botStartChat();

    ?>

botTransfer

Transfere o antedimento humano. O tipo do canal é identificado automaticamente.

Funcao:
botTransfer($cd_servico=0, $cifra_servico=0)

Parametros:
$cd_servico - Int ->
   Codigo do servico que será enviado. Ex: 545
   Envia para o serviço caso preenchido.
$cifra_servico - Int ->
   Cifra do servico que será enviado. Ex: 1101
   Verifica o formulário de serviço.
     Local json: /home/extend/servico/1101/1101.json.
     Obs: Para ativar apenas o formulário de serviço, basta configurar o $cd_servico=0. 

Exemplo: 
    <?php
        header('Content-Type: text/html; charset=utf-8');
        header("Access-Control-Allow-Origin: *");
        header("Access-Control-Allow-Headers: Content-Type");
        include dirname(__FILE__).'/functionsBot.php';
        error_reporting(1);

        $id=$argv[1];
        $telefone=$argv[2];
        $assunto=$argv[3];


        botStartChat();
        sleep(5);

        botFinishChat();

    ?>

botInsertButtom

Envia Menu de opção para o usuário

Funcao:
botInsertButtom($btn,$wpp_prefix=1)

Parametros:
* $btn - Array ->
   O primeiro indice do array será sempre o título.
   Caso não tenha título, configurar como ""
$wpp_prefix - Int ->
   Gera numeração na frente de cada opção quando está no Whatsapp
   Ativado por padrão. Configurar 0 para retirar

        $menu = array("Escolha uma das opções",
                      "Financeiro",
                      "Comercial");

        botFinishChat();

    ?>

botInsertPassword

Envia campo no formato de senha. Campo com caracteres mascarados.
A Função trava o campo de digitação do usuário, forçando-o a digitar no campo de senha.
Para destravar o campo de digitação deve ser enviado o comando 'botInsertMsg("Ö!Ö")' apos o uso.

IMPORTANTE: Função não está disponível para Whatsapp. Caso utilizado no Whatsapp, será enviado uma mensagem normal.

Funcao:
botInsertPassword($msg)

Parametros:
* $file - String 

Exemplo: 
    <?php
        header('Content-Type: text/html; charset=utf-8');
        header("Access-Control-Allow-Origin: *");
        header("Access-Control-Allow-Headers: Content-Type");
        include dirname(__FILE__).'/functionsBot.php';
        error_reporting(1);

        $id=$argv[1];
        $telefone=$argv[2];
        $assunto=$argv[3];


        botStartChat();

        botInsertPassword("Por favor, digite a sua senha");

        $text = botGetFull();
        botInsertMsg("Ö!Ö")


        botFinishChat();

    ?>


botGetFull

Captura a digitação do usuário.

Configuração padrão em caso do uso da função sem parâmetros.
   60 segundos
   3 repetições
   Mensagem de repetição: "Oi você ainda está por ai? Por favor escolha uma das opções"
   Mensagem da 3° repetição: "Agradecemos seu contato!"


Em caso de envio de áudio pelo usuário no Whatsapp, a função detecta se a url publicada está na mensagem e traduz automaticamente o áudio para texto.
Requisitos para funcionalidade:
   * ffmpeg instalado
   * Arquivo google_stt.py no diretorio: '/home/extend/scripts/google_stt.py'
   * Chave da Google TTS com nome de "key.json" no diretorio: '/root/google-cloud-sdk/key.json'

Funcao:
botGetFull($timeout=60,$repeticao_max=3,$repeticao_msg_rep="",$repeticao_msg_fim="")

Parametros:
$timeout - Int ->
   Tempo total de espera em segundos.
$repeticao_max - Int ->
   Quantidade de repetições.
$repeticao_msg_rep - String/Array ->
   Mensagem que será enviada após cada repetição.
   Em caso de Array, a cada repetição será enviado cada índice do Array
$repeticao_msg_fim - String ->
   Mensagem que será envida na última iteração.

Retorno:
String com texto digitado pelo usuário.

Exemplo 1:
    <?php
        header('Content-Type: text/html; charset=utf-8');
        header("Access-Control-Allow-Origin: *");
        header("Access-Control-Allow-Headers: Content-Type");
        include dirname(__FILE__).'/functionsBot.php';
        error_reporting(1);

        $id=$argv[1];

        $assunto=$argv[3];

        $menu = array("Escolha uma das opções",
                      "Financeiro",
                      "Comercial");

        botInsertButtom($menu);

        //variavel $text recebe texto digitado pelo usuário

        botFinishChat();

    ?>

botFaq

Utiliza a aplicação da Sophia, e retorna o valor do dicionário.
A função registra automaticamente os dados no banco quando as variaveis '$bot_ura', '$bot_menu', '$bot_opcao' estiverem preenchidas no fluxo.

IMPORTANTE: Necessário ter a aplicação da Sophia para o funcionamento.
Diretório padrão: /home/extend/sophia/

Funcao:
botFaq($dic_sophia, $duvida, $assunto="")

Parametros:
* $dic_sophia - String ->
   Caminho completo do diciónario
* $duvida - String ->
   Texto com a mensagem que será enviada para o dicionário.
* $assunto - String ->
   Assunto relacionado. Registra automaticamente nos relatórios quando configurado. 

Exemplo: 
    <?php
        header('Content-Type: text/html; charset=utf-8');
        header("Access-Control-Allow-Origin: *");
        header("Access-Control-Allow-Headers: Content-Type");
        include dirname(__FILE__).'/functionsBot.php';
        error_reporting(1);

        $id=$argv[1];
        $telefone=$argv[2];
        $assunto=$argv[3];

        $sophia_dic = "/home/extend/sophia/sophia.ai";

        botStartChat();
        botInsertMsg("Saudações. Bem vindo ao nosso Bot");

        $menu = array("Escolha uma das opções",
                      "Financeiro",
                      "Comercial");

        botInsertButtom($menu);

        $text = botGetFull();
        $faq_results = botFaq($sophia_dic, $text);

        botFinishChat();

    ?>
